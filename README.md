# TwistPuzzle

**TwistPuzzle** - це гра-головоломка, де гравець повинен використовувати особливості кожного рівня та дістатися до його кінця.

# Інструкція для запуску
### ВАЖЛИВО: вам знадобиться Unity Editor так, як готової збірки покищо немає.
1. Завантажити zip-файл з Gitgub.
2. Розпакувати в потрібне вам місце.
3. За допомогою Unity Hub відкрити цей проект.
4. Після відкриття вибрати його в доступних проектах.
5. Зачекати поки відкриється Unity Editor та запустити його.
